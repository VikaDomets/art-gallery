# VA Gallery — Modern Art Space Platform

VA Gallery — це сучасний веб-застосунок для художніх галерей, який дозволяє переглядати поточні виставки, ознайомлюватися з каталогом робіт та керувати користувацьким досвідом.

## Live Demo


## Ключові особливості
- **Global State Management:** Використання React Context API для керування даними виставок та станом авторизації.
- **Client-side Routing:** Навігація між сторінками без перезавантаження за допомогою React Router.
- **Responsive Design:** Повністю адаптивний інтерфейс (Mobile-first), що коректно відображається на смартфонах, планшетах та десктопах.
- **Dynamic UI:** Плавні анімації переходів між сторінками та елементами пагінації.
- **Mock Data Integration:** Система налаштована на роботу з локальними даними для забезпечення стабільної демонстрації функціоналу.

## Технологічний стек
- **Frontend:** React.js, Vite
- **Styling:** CSS Modules, BEM methodology
- **Navigation:** React Router DOM
- **State Management:** React Context API
- **Deployment:** Vercel
